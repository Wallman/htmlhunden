# README

## Get up and running
För att undvika dependecy och filnamns-konflikter så bygger vi nu projektet genom Grunt.

### Installera Grunt
`$ npm install -g grunt-cli --save dev`

### Hämta dependencies
`$ npm install`

## Tasks
- `$ grunt jade` Kompilerar alla jade-filer i mappen /src
- `$ grunt watch` Kör ovan task vid ändringar i /src-mappen
- `$ grunt` Alias för ovan

Med andra ord. Med Grunt installerat. Kör helt enkelt bara `grunt` och koda sen på.
